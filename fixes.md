> # Pathway v0.1.1
>
> ## AI
>
> - Auto generation now opens a modal showing AI tasks with checkboxes — select only the ones you want before adding
>
> ## Visuals
>
> - Lightness bar range extended 15% darker (35–80% instead of 50–80%)
> - Selected tasks in daily generation no longer show strikethrough
> - Scrollbar removed from main content area
>
> ## Settings
>
> - Daily reset time is now configurable (default 4 AM)
> - Nav bar order is configurable with up/down arrows — Done is always last but can be hidden
>
> ## QOL
>
> - Pressing Enter to create a new task now defaults parameters to 1 1 1 instead of copying the row above
> - Fixed bug where tasks with the same name in daily generation would all tick at once
>
> ## Trackers page
>
> - New Trackers page backed by a single markdown file (_Trackers.md)
> - Each tracker has a name, current value, goal, and importance
> - UI shows a progress bar, percentage, fraction, and — / + buttons
> - Trackers sorted by importance, completed trackers highlight green
> - Markdown edit mode accessible from the page header
